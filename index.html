How to use EpiMap

Quick Start

  1. Click Load Demo Data to see the tool immediately.
  2. Or upload a zipped shapefile and a CSV.
  3. Set the join key — the field linking shapefile to CSV.
  4. If your CSV is granular (long format) — e.g. one row per county × age × gender — the tool will detect this automatically and show aggregation options: filter by age range, filter by category (gender, year…), choose sum/mean/count etc.
  5. Choose a variable, classification, and color palette in the Choropleth Layer panel.
  6. Optionally enable Proportional Symbols or Bivariate mode.
  7. Add title, legend, north arrow in the right panel. Export PNG when ready.

CSV Formats Supported

  • Wide / pre-aggregated — one row per geographic unit, multiple columns. Joined directly.
  • Long / granular — multiple rows per unit (e.g. by age, gender, year). Tool detects this and prompts for aggregation: filter by age range or category, then sum/mean/median/count.

Classification Schemes

  • Jenks — natural breaks, minimizes within-class variance; best for most epi data
  • Quantile — equal features per class; good for skewed distributions
  • Equal Interval — equal data range; intuitive but sensitive to outliers
  • Std Deviation — classes relative to mean
  • Manual — define your own breakpoints

Colorblind-Safe Palettes

Palettes marked ✓CB (Viridis, Cividis, Plasma, YlGnBu) are safe for all major types of color vision deficiency.
