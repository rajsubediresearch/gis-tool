How to use EpiMap

Quick Start

  1. Load Demo Data to see the tool in action immediately.
  2. Or upload a zipped shapefile (.zip containing .shp, .dbf, .prj) and a CSV with your attribute data.
  3. Set the join key — the field that links your shapefile to your CSV (e.g., state name, FIPS code).
  4. Choose a variable, classification scheme, and color palette in the Choropleth Layer panel.
  5. Optionally enable Proportional Symbols or Bivariate mode for multi-variable mapping.
  6. Add a title, legend, north arrow using the right panel.
  7. Export PNG when ready.

Classification Schemes

  • Jenks (Natural Breaks) — minimizes within-class variance; best for most epidemiological data
  • Quantile — equal number of features per class; good for skewed distributions
  • Equal Interval — equal data range per class; intuitive but sensitive to outliers
  • Std Deviation — classes relative to mean; useful for showing deviation from average
  • Manual — define your own break points

Colorblind-Safe Palettes

Palettes marked ✓CB (Viridis, Cividis, Plasma, YlGnBu) are safe for all major types of color vision deficiency.

Shapefile Format

Zip your .shp, .dbf, .prj (and optionally .shx) into a single .zip file. The tool reads WGS84 (EPSG:4326) coordinates. Most shapefiles from GADM, Natural Earth, Census TIGER are already in WGS84.
